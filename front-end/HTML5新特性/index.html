HTML5新特性

发表于 2020-06-30 | 分类于 前端 |
HTML5带来了很多新的api和元素

HTML5是HTML的第五个版本，它是互联网的下一代标准，上一版是HTML4.01
HTML4基于SGML，因此HTML4.01中的doctype需要对DTD进行引用，HTML5不基于SGML，因此不必加入DTD

DOCTYPE标签

<!DOCTYPE>标签需要位于html5文档的第一行

doctype 声明不属于 HTML 标签，它是一条指令，告诉浏览器编写页面所用的标记的版本。在所有 HTML 文档中规定 doctype 是非常重要的，这样浏览器就能了解预期的文档类型。

新特性

标签

  • HTML5引入了很多语义标签比如<section>,<header>,<article>,<summary>

  • HTML5的表单元素更加丰富，input的类型多了color, week,range等等以前需要插件来实现的

  • HTML5增加了<svg>,<canvas>元素，可以用于绘图

  • 增加了<audio><video>显示多媒体类型数据

API

  • 引入了Geolocation 的API可以通过GPS或者网络信息来提供定位信息
  • Drag & Drop
  • 提供了新的客户端存储方式（localStorage和sessionStorage）和Web database（Web SQL和indexed DB)
  • 引入了web worker从而可以实现JS的多线程
  • 引入了SSE
  • 对于history对象引入了pushState和replaceState方法，可以修改url同时不跳转
