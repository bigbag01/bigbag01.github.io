VSCode 插件

发表于 2018-12-12 | 分类于 工具 |
在同学安利下开始用vscode，这篇blog是关于一些插件和配置的记录

Live Server

简介： 静态服务器，需要访问本地文件的时候好用。安装之后会在vscode右下角有一个 Go Live的按钮，点击后就可以启动，默认的端口是5500。

Debugger for Chrome

简介： 算是前端利器吧，配合chrome可以直接在vscode里对前端的文件进行debug。双屏的时候，一个看页面效果，一个开调试还是非常棒的，不用像之前在chrome里调试好了再去文件里找对应的地方修改。

TODO Hightlight & Todo Tree

简介： 明显这两个都是为todo准备的插件，前者的作用是让任何注释中的TODO: 、FIXEME:都会被高亮，后者在vscode左侧功能栏增加了一个树形按钮，点开后会有目录列出工作目录下所有标了todo或者fixme的文件和这些注释所在位置。非常建议这两个插件一起用。
