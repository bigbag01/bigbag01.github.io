网络通信-URL,URI

发表于 2020-02-18 | 分类于 网络通信 |

URL

统一资源定位符 Uniform Resource Locator，访问web页面的地址，表示资源地点，网络上的位置。

  • URL实例符合规则scheme://host.domain:port/path/filename
    scheme通常可以是http,https,ftp,file等，http默认port为80
  • URL只能包含ASCII字符，对于非ASCII字符，使用 “%” 其后跟随两位的十六进制数来替换。
  • URL不能包含空格

URI

统一资源标识符 Uniform Resource Identifier，表示某一互联网资源。URI在某一个规则下可以把一个资源独一无二的标识出来。URI有两种形式，URL或者URN。
一个标准的URI包含 协议名+域名+路径

URN

标识书籍的ISBN系统就是URN形式，URN ( Uniform Resource Name)是基于某名字空间通过名称指定资源的URI。人们可以通过URN来指出某个资源，而无需指出其位置和获得方式。

区别

URL是一种具体的URI，它是URI的一个子集，它不仅唯一标识资源，而且还提供了定位该资源的信息。URI 是一种语义上的抽象概念，可以是绝对的，也可以是相对的，而URL则必须提供足够的信息来定位，是绝对的。
