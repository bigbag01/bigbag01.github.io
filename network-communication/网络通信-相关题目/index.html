网络通信-相关题目

发表于 2020-02-18 | 分类于 网络通信 |

  1. 计算机网络按作用范围可以分为广域网、 城域网、 局域网

  2. IP数据报的首部的最大长度是（ ）个字节？

    A. 20 B. 40 C 60
    答案： C
    解析：IP数据包首部固定20字节，可变40字节，因此长度在20-60之间，最大是60字节。

  3. 应用程序ping收到的是（ ）
    A. TCP应答 B.ICMP请求 C.ICMP应答 D.TCP差错报告
    答案：C
    解析：ping用于测试网络是否可达，ICMP就是一个“错误侦测与回报机制”，其目的就是让我们能够检测网路的连线状况﹐也能确保连线的准确性。

  4. 完成远程登录的TCP/IP协议是 （ telnet ）
    解析： telnet是Internet远程登录服务的标准协议和主要方式

  5. 某公司网络的地址是 202.110.128.0/17 ，下面的选项中， （） 属于这个网络。
    A 202.110.44.0/17
    B 202.110.162.0/20
    C 202.110.144.0/16
    D 202.110.24.0/20
    答案： B
    解析：给定的地址的掩码是 255.255.128.0，A的是255.255.0.0，B是255.255.160.0，C是255.255.0.0，D是255.255.16.0.
    ACD都无法覆盖到128，所以是B。牛客解答
