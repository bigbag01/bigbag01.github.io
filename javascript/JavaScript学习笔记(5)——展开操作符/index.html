Javascript学习笔记(5)——展开操作符

发表于 2020-02-05 | 分类于 javascript |
ES6中引入了展开操作符，`...` 可以将可迭代对象转为`,`分隔的参数序列

用于数组

用于类似数组的对象

可迭代对象

var a=[…Array(n).keys()] 可以得到 [0,1,…,n-1]的一个数组，Array(n).keys()原本是一个可迭代对象，通过…展开得到数组

类对象数组

var arr=[…document.getElementsByTagName(‘a’)],可以将类对象数组转为真正的数组

字符串

var strArray = [...'abc’] 得到[‘a’,’b’,’c’],相当于Array.from('abc')或者'abc'.split('')

具有iterator接口的对象

Set, Map

generator用

用于对象

  • …可以复制对象中的属性，但只会做浅拷贝，不会递归深度拷贝所有属性
  • …可以做对象合并,相当于Object.assign({},obj,newObj)

函数的rest参数

可以用…rest的数组替代arguments的方式来获取剩下未知的参数。
