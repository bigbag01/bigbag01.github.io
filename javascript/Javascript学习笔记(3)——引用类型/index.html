发表于 2017-11-13 | 分类于 javascript |

引用类型

引用类型的值是引用类型的一个实例。引用类型有时也被称为对象定义。

ECMAScript中提供了很多原生引用类型，

  • 内置对象：Object, Array, Date, RegExp, Function

  • 基本类型的包装类型 Boolean, Number, String

  • 单体内置对象：Global, Math

  • ES6新增：Promise, Map, Set, ArrayBuffer, TypedBuffer, DataView……

    同时ES6也对原有内置对象进行了扩展

Function

函数名实际上是一个指向函数对象的指针，不会与某个函数绑定。

创建Function实例的方式

解析器在向执行环境中加载数据时，会先读取函数声明，并使得它在执行任何代码之前可用，而函数表达式则按上下文顺序加载。所以，通过“函数声明提升”，即使声明函数的代码在调用它的代码之后，js引擎也可以把它提升到顶部，如果写成函数表达式，则不可以。

函数内部属性
arguments和this

函数内部有两个属性

  • arguments的主要用途是保存函数参数，但它还有一个名叫callee的属性，callee是一个指针，指向拥有这个arguments对象的函数。
  • this的作用和C++，Java中类似，它引用的是函数执行的环境对象。当在浏览器的全局作用域中调用函数时，this对象引用的就是window。

函数有caller属性，可以获得当前函数执行环境代码，如果在全局作用域中，则获得null

length和prototype

每个函数包含两个属性：length，prototype。

  • length表示希望接收的命名参数的个数
  • prototype 是保存引用类型所有实例方法的地方。
apply和call

每个函数包含两个非继承而来的方法： apply(), call()。它们的用途都是在特定的作用域中调用函数，实际上是设置函数体内this的值。

  • apply接收两个参数，一个是在其中运行函数的作用域，一个是参数数组。
  • call的第一个参数是在其中运行函数的作用域，之后的参数则是函数参数的逐个列举。
  • bind 另外有一个bind方法，创建一个函数实例，其this值被绑定到传给bind的参数上。

  • 基本包装类型提供了操作相应基本类型值的便捷方法。

  • 使用new调用基本包装类型的构造函数和直接调用同名的转型函数是不一样的。
