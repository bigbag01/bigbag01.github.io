浮点表示不精确

js中数值类型只有Number，运算结果如果是浮点数就会直接以浮点数形式给出，而不会自动转换为整型。然而计算机中浮点数的表示是不精确的。

1.0-0.9 的结果并不是0.1，因此如果做判断 1.0-0.9 == 0.1 结果会是false。
