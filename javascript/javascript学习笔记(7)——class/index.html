Javascript学习笔记(7)——class

发表于 2020-02-11 | 分类于 javascript |
ES6中引入了class关键字，可以像其他语言一样的写法来构造JS中的“类”

class

ES6引入了class的写法，可以看成是另一种构造函数

  • 虽然class表示的也是一个function，但是使用这种function的时候必须要加new关键字，否则会报错。
  • 除非显示地写this.xx=yy，否则属性和方法都是定义在原型上的。

扩展类成员

向一个类添加成员，其实是向它的prototype中添加成员。

读取类成员

以class这样写法定义的类的成员都是不可枚举，不能由Object.keys(Creature)获取

class表达式

表达式定义

class有类似函数表达式的定义形式

  • 这个类的名字是MyClass, Me只有在内部可以使用
  • 如果内部不用到，也可以省略 Me

立即执行

无变量提升

class没有变量提升

static

  • 类方法前面加了static关键字成为静态方法就不会被实例继承，可以直接通过类来调用
  • 类的静态方法和实例方法可以重名
  • 静态方法可以被子类继承

私有

可以通过symbol作为私有方法名和属性名，然后把symbol值放在模块内部，这样外面很难获取到。

getter/setter

  • class内部可以用get和set对某个属性设置存取函数，拦截其存取行为

  • 他们被定义在属性的描述对象上

继承

  • 用extends关键字继承

  • 子类如果自己定义了constructor，其中必须先用super()来调用父类的constructor

  • 可以用getPrototypeOf来获取父类

  • 子类的__proto__指向父类

  • 子类的prototype.__proto__指向父类的prototype
