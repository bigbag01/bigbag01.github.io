Javascript学习笔记(16)——event loop

发表于 2020-02-18 | 分类于 javascript |
js的事件循环，异步的本质

浏览器的事件循环

  1. 执行全局script的同步代码，等执行完调用栈stack会清空
  2. 从微任务队列取出队首的回调任务，放入调用栈stack，执行后微队列长度减一
  3. 不断取微任务放到stack中执行，直到微任务队列清空，stack也清空
  4. 从宏任务队列队首取回调任务放入调用栈stack，宏队列长度减一
  5. 不断取宏任务放入stack执行，直到stack为空。
  6. 不断重复3-5

重点

  1. 宏队列一次只取一个，取完先去检查微队列有没有任务，没有再取自己的下一个
  2. 微队列会不断取出来执行直到为空
  3. UI渲染任务会紧跟着微队列的清空

宏任务队列

浏览器的宏任务包括

  • setTimeout
  • setInterval
  • requestAnimationFrame
  • I/O
  • UI 渲染

微任务队列

浏览器微任务包括

  • promise
  • object.observe
  • MutationObserver
  • 监听DOM的变化
