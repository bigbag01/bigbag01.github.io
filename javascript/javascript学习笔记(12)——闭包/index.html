Javascript学习笔记(12)——闭包

发表于 2020-02-16 | 分类于 javascript |
闭包真是js里经常遇到的一个东西，算是js的语言特性，也很容易考到。
  • 闭包是让你能访问一个隐藏起来的变量的方法
  • JS中面向对象没有public和private的限定，闭包可以用来模拟private

在循环中使用闭包

上面这个会出现是因为i的作用域是在func中的，它在闭包中被保存，所以当调用func再去读i的时候，它的值是最新的3；
如果要修改，可以直接把循环体中的var i=0改成let i=0,它的作用域就被限制在循环体内，这个值就会定格;

清除闭包

闭包的生命周期是跟随包含它的函数，如果要手动清除，就把不需要的函数名设为null
