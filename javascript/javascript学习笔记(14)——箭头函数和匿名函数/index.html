Javascript学习笔记(14)——箭头函数和匿名函数

发表于 2020-02-17 | 分类于 javascript |
箭头函数的this作用域很特别；匿名函数的立即执行特性和闭包。

this指针区别

  • 普通函数和匿名函数的this通常指向调用它的对象，匿名函数的this通常都是指向window
  • 箭头函数内部的this指向定义这个函数时的作用域的this

参数

箭头函数内部没有arguments对象，可以通过解构赋值或者展开操作符的方式来获取参数列表

构造函数

箭头函数不能当做构造函数
