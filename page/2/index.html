引用类型

引用类型的值是引用类型的一个实例。引用类型有时也被称为对象定义。

ECMAScript中提供了很多原生引用类型，

  • 内置对象：Object, Array, Date, RegExp, Function

  • 基本类型的包装类型 Boolean, Number, String

  • 单体内置对象：Global, Math

  • ES6新增：Promise, Map, Set, ArrayBuffer, TypedBuffer, DataView……

    同时ES6也对原有内置对象进行了扩展

发表于 2017-11-08 | 分类于 javascript

变量

​ ECMAScript 变量有两种不同数据类型的值：基本类型值（简单的数据段）和引用类型值 （可能由多个值构成的对象）。

​ 基本数据类型是按值访问的，可以操作保存在变量里的值。

​ 引用类型的值是保存在内存里的对象，是按引用访问的。操作对象时，是操作对象的引用。

​ 注意： ECMAScript和其他语言不同，这里的字符串是基本类型而非引用类型
