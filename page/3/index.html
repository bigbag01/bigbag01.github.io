变量

​ ECMAScript 变量有两种不同数据类型的值：基本类型值（简单的数据段）和引用类型值 （可能由多个值构成的对象）。

​ 基本数据类型是按值访问的，可以操作保存在变量里的值。

​ 引用类型的值是保存在内存里的对象，是按引用访问的。操作对象时，是操作对象的引用。

​ 注意： ECMAScript和其他语言不同，这里的字符串是基本类型而非引用类型

Javascript学习笔记(1)——基本概念

发表于 2017-10-30 | 分类于 javascript
写过一些前端页面，对前端知识有浅显的了解，开始学习js的更多知识。这个系列是以读《JavaScript高级程序设计》为主，查看其他相关资料的笔记。

插入javascript脚本

  • 建议用<script src="some URL"></script>引用外部js脚本而非将js代码嵌入html文件中，原因主要有可维护性好，且可缓存。
  • 所有script标签在没有defer和async属性时，按照它们在页面中出现的顺序依次被解析。
  • 由于浏览器会先解析script标签引用的代码，然后再解析之后的内容，因此，一般应该把script标签放最后，</body>之前。
  • 使用defer属性可以让脚本再document加载完成之后再执行
  • 使用async属性可以表示当前脚本不必等待其他脚本，也不阻塞文档加载。使用了async属性的脚本执行顺序无法保证

OneTab

简介： tab页收集工具

优点：

  1. 点击插件图标就可以将当前chrome浏览器中打开的页面都收集起来。有的网页暂时不需要浏览又不想放入收藏夹的时候就可以用它。

好用的软件分享——windows平台

发表于 2017-04-17 | 分类于 工具
我经常会在网上看大家推荐一些软件，对于一些感兴趣的也会去试用。这一篇博客是关于windows平台下，我用过觉得不错的软件，有的很有名，有的还很冷门。这些软件大多数都是让我觉得可以帮助提高工作学习效率的。

typora

简介 ：markdown编辑器。typora是windows下我目前用过的感觉最棒的markdown编辑器，现在博客基本都是在这里编辑。

优点：

  1. 所见即所得，行内预览，书写感觉非常流畅。
  2. 插入图片、表格、数学公式等都非常方便，有菜单栏可选，熟悉了以后也可以直接用快捷键。
  3. 直接附带转为pdf和html的功能，如果配合pandoc一起用，还可以直接将markdown转化为docx、epub等格式
  4. 配套theme可定制，程序员们可以用css写一套喜欢的主题
  5. 因为是用js构建的，界面其实是内嵌了chromium浏览器，所以typora是跨平台的。
  6. 免费

缺点：

  1. 单机，无同步功能。然而我把笔记都放到博客上了，直接打开网址就可以看了
  2. 页面为js渲染，打开比较慢
