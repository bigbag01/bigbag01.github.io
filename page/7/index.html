OneTab

简介： tab页收集工具

优点：

  1. 点击插件图标就可以将当前chrome浏览器中打开的页面都收集起来。有的网页暂时不需要浏览又不想放入收藏夹的时候就可以用它。
  2. 收集起来的网页按日期分开，可以直接删除也可以一键恢复

缺点：

  1. 有时候只是想点开看之前收起的页面，但是当前的就都一起被收起了，重新加载会花时间

好用的软件分享——windows平台

发表于 2017-04-17 | 分类于 工具
我经常会在网上看大家推荐一些软件，对于一些感兴趣的也会去试用。这一篇博客是关于windows平台下，我用过觉得不错的软件，有的很有名，有的还很冷门。这些软件大多数都是让我觉得可以帮助提高工作学习效率的。

typora

简介 ：markdown编辑器。typora是windows下我目前用过的感觉最棒的markdown编辑器，现在博客基本都是在这里编辑。

优点：

  1. 所见即所得，行内预览，书写感觉非常流畅。
  2. 插入图片、表格、数学公式等都非常方便，有菜单栏可选，熟悉了以后也可以直接用快捷键。
  3. 直接附带转为pdf和html的功能，如果配合pandoc一起用，还可以直接将markdown转化为docx、epub等格式
  4. 配套theme可定制，程序员们可以用css写一套喜欢的主题
  5. 因为是用js构建的，界面其实是内嵌了chromium浏览器，所以typora是跨平台的。
  6. 免费

缺点：

  1. 单机，无同步功能。然而我把笔记都放到博客上了，直接打开网址就可以看了
  2. 页面为js渲染，打开比较慢

线程

线程是程序中一个单一的顺序控制流程。进程内一个相对独立的、可调度的执行单元，是系统独立调度和分派CPU的基本单位指运行中的程序的调度单位。在单个程序中同时运行多个线程完成不同的工作，称为多线程。一个标准的线程由线程ID，当前指令指针（PC），寄存器集合和堆栈组成。

进程

进程 （Process）是计算机中的程序关于某数据集合上的一次运行活动，是系统进行资源分配和调度的基本单位，是OS结构的基础。在早期面向进程设计的计算机结构中，进程是程序的基本执行实体；在当代面向线程设计的计算机结构中，进程是线程的容器。程序是指令、数据及其组织形式的描述，进程是程序的实体。

二者区别

​ Arrays 继承自java.lang.Object,它属于java.util工具包，它是java中的工具类。它包含了很多直接操作数组的方法，比如数组的排序、搜索等。其中还包括一个可以将数组作为列表来查看的静态工厂 。这个类中方法全都是static的，可以直接用类名.方法来调用。

​ Array 数组类，它是java中最基本的一个存储结构。它提供了动态创建和访问java数组的方法。一个Array中的元素类型必须相同。Array效率高，但容量固定无法动态改变。array.length只能得到array的容量，无法判断其中实际元素个数
